Ejercicio 1: Variables y operaciones aritméticas básicas 1. Crea una variable llamada a y asígnale el valor 10. 2. Crea otra variable llamada b y asígnale el valor 5. 3. Suma las dos variables y guarda el resultado en una variable llamada suma. 4. Resta b a a y guarda el resultado en una variable llamada resta. 5. Multiplica a por b y guarda el resultado en una variable llamada multiplicacion. 6. Divide a entre b y guarda el resultado en una variable llamada division. 7. Muestra todos los resultados (suma, resta, multiplicacion, division) en la consola.
